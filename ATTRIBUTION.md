## Adding new clips

When contributing a clip via `data-pipeline/curated_downloads.py` or by
dropping a file into `data/sounds/<category>/`, you MUST:

1. Verify the source publishes the file under a free license that
   permits redistribution (CC0, CC-BY, CC-BY-SA, public domain).
2. Add a row in this file with the source URL and license.
3. If the license is CC-BY-NC-* or any non-redistributable variant, do
   not commit the file — keep it locally.

## Why CC-BY-NC-SA is included for some birds

`bird-american-robin` is sourced from Xeno-canto via Wikimedia Commons
and is licensed CC-BY-NC-SA-3.0. The "NC" clause restricts **commercial**
distribution. Auralis is a non-commercial portfolio project; the clip
is included with full attribution and the same NC-SA license is
preserved for downstream users. If you intend to use Auralis
commercially, replace this clip with an unrestricted alternative.
